# Macro outcome

The daily macro-window outcome table produced by `macro_outcomes.py` from canonical minute
bars: per-date direction (bullish / bearish / neutral), range, and extreme-timing fields for
the [[macro-window]]. The baseline against which experiments measure separation — direction is
close to a coin-flip (~49% bullish / ~49% bearish).
