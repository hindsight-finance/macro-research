# Barrier context (15:50 barrier)

Which side of an early macro **barrier** (e.g. the 15:50 minute high/low) price touches first,
and how it behaves afterward — wrong-side closes, worst wrong-side distance, time/share spent
wrong-side. Joined with [[anchored-vwap]] side to compare barrier-only vs VWAP-only vs
barrier+VWAP context. Output: `outputs/nq_macro_1550_barrier.parquet`.
