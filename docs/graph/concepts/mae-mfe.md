# MAE / MFE (excursion)

**Maximum Adverse Excursion** and **Maximum Favorable Excursion** — the worst and best
unrealised moves over a window relative to an entry/anchor. Used to profile how far price
travels for/against a side, especially for [[fair-value-gap]] outcomes and delta-conditioned
profiles.
