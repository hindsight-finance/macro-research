# Fair value gap (FVG)

A price imbalance / gap left by a directional move (three-bar imbalance). The workspace studies
FVG **formation, alignment, excursion, success context, and volume** around the [[macro-window]]
and in the PM hour, often joined with [[volume-delta]] to test which side dominates a gap.
