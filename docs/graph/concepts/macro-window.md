# Macro window

The market-close **macro window, 15:50–15:59 ET** (with `POST` = 16:00–16:10 ET as the
follow-through). The central object of this workspace: its range, direction, timing, tick
microstructure, volume delta, FVG behaviour, liquidity, and trend/regime context.

Window labels (see [[session-windows]]): `H3PM` = 15:00–15:49, `MACRO` = 15:50–15:59,
`POST` = 16:00–16:10. Daily outcomes summarised in [[macro-outcome]].

**Variants.** [[macro-open-1550]] · [[macro-close-1559]]
**Related.** [[session-windows]] · [[time-handling]] · [[macro-outcome]]
**Used by.** nearly every experiment — see [[Concept Map]].
